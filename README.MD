## Setup

Install Node.js + npm by following the platform-specific instrutions [here](https://nodejs.org/en/).
On the Unix command line, `cd` into `front-end` and run `npm install`.

To run the webapp, just run the command `npm run web`.
Android and IOS versions of the app require compatible devices or emulators, as well as external software (potentially).
